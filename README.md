Computer Science student at IUBAT (Dhaka) focused on computer vision, efficient machine learning, and AI systems.

---

## Featured projects

Training-free compression pipeline reducing 3D Gaussian Splatting model size ~74% across three scenes with minimal quality loss (~1 dB PSNR).

**[splat-slim](https://github.com/Daceyyreal/splat-slim)** — Training-free post-processing that shrinks trained 3DGS models. Prune, reduce spherical harmonics, and quantize any `.ply` for large size cuts at near-lossless quality. Companion code to my thesis.

**[splat-lab](https://github.com/Daceyyreal/splat-lab)** · **[live demo →](https://daceyyreal.github.io/splat-lab/)** — Interactive, in-browser 3DGS compression. Load a scene and watch file size and quality change in real time as you prune, drop SH bands, and quantize. The visual companion to splat-slim.
